# Chuck Dreamer

## Installation

Requires Python 3.12+. Uses [uv](https://github.com/astral-sh/uv) for package management.

```bash
uv sync
```

For development dependencies:

```bash
uv sync --dev
```

## Usage

```bash
python main.py [--config PATH] [--verbose] COMMAND [OPTIONS]
```

See the `--help` flag for more details.
